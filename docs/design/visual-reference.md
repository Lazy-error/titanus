# Visual Reference

## Target Visual Layout

The project uses an agreed **target visual reference** for:

- world rendering
- pseudo-3D readability
- navigation readability
- height composition
- map editor goals

This reference establishes the following scene principles:

- top-down / pseudo-3D presentation
- readable multi-level terrain
- cliffs and plateaus as clear height boundaries
- central open combat zone
- elevated tactical positions
- high platforms suitable for totems, minibosses, and control points
- visually readable jump / drop / choke-point scenarios

## Important Note

The visual reference is a **structural target**, not a requirement to literally copy one screenshot.
